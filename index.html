Introduction



Chatbot System Architecture

This project showcases a chatbot system integrated with various services. It leverages Google Cloud's Dialogflow as the core chatbot engine, enabling natural language interactions.

Here's a breakdown of the key components:

  • Knowledge Base (Top Right): Stores information and data the chatbot uses to respond to user queries.
  • Fulfillment Services (Bottom Right):
    • Google Cloud SQL: Managed relational database service for storing chatbot-related data.
    • Cloud Run: Serverless platform to execute custom code triggered by the chatbot.
  • Integrations (Left Side):
    • Telephony and Web Interface: Enables the chatbot to interact with users through phone calls and web interfaces.
    • Twilio (Bottom Left): Facilitates WhatsApp integration for the chatbot.

This architecture demonstrates how a comprehensive chatbot system can interact with users across diverse channels while leveraging various data sources and services.
